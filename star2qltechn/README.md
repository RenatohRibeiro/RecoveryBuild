# OrangeFox S9 - Starqltechn

## How to build

```bash
# download script
$ wget https://raw.githubusercontent.com/RenatohRibeiro/RecoveryBuild/master/star2qltechn/auto.sh

# Change the script permission
$ chmod +x ./auto.sh

# Build
$ ./auto.sh

# Or do it all in one line
$ wget https://raw.githubusercontent.com/RenatohRibeiro/RecoveryBuild/master/star2qltechn/auto.sh && sudo chmod +x ./auto.sh && ./auto.sh
```
